# Healthcare-data-breach-analysis
Data analysis project in R examining healthcare data types and their impact on affected individuals.
# What this project is
I built this project to practice working with real data in R. I wanted to see which types of healthcare data breaches affect the most people.
# Question
Which types of healthcare data breaches result in the largest impact on individuals?
# Hypothesis
I thought that digital breaches (like servers or computers) would affect more people than paper-based ones.
# What I did
- Cleaned and simplified the dataset using R
- Grouped different types of breaches into broader categories
- Compared how many people were affected by each type
- Looked at averages, medians, and maximum values
# What I found
- “Loss” had the highest average impact
- “Theft” happened the most but didn’t affect the most people per incident
- The data was highly skewed; a few big breaches affected a lot of people
# Conclusion
My hypothesis was partially correct. Some digital breaches had a high impact, but other types (like loss) also affected a lot of people. Overall, the high-impact incidents are not limited to digital systems.
